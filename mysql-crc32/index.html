September 08, 2019
#mysql #crc32

If need to calculate the CRC32 of a string, using mysql:

$ mysql -e "SELECT CRC32('127.0.0.1')"
+--------------------+
| CRC32('127.0.0.1') |
+--------------------+
|         3619153832 |
