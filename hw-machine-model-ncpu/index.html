hw.(machine|model|ncpu)

June 13, 2018
#freebsd #ncpu #sysctl

To get an overview of the system:

# sysctl -a | egrep -i 'hw.machine|hw.model|hw.ncpu'
hw.machine: amd64
hw.model: Intel(R) Xeon(R) Silver 4110 CPU @ 2.10GHz
hw.ncpu: 16
hw.machine_arch: amd64
