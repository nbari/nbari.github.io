dropbox-cli No module named 'gi'

March 15, 2025

When trying to install dropbox-cli on a Linux system, you may encounter the following error:

../dropbox-cli/src/nautilus-dropbox-2024.04.17/build_dropbox.py", line 2, in <module>
    import gi
ModuleNotFoundError: No module named 'gi'
==> ERROR: A failure occurred in build().

Try to install pygobject:

pip install pygobject

Then try to install dropbox-cli again:

paru -Syu dropbox-cli
