Git SSH

August 29, 2018
#git #ssh #bare

To create a git repository and access to it via ssh:

$ ssh your.host
$ mkdir my-new-repo
$ cd my-new-repo
$ git --bare init

To access your repo (clone it):

$ git clone ssh://user@your.host:2222/~user/my-new-repo
