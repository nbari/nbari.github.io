sockstat

May 13, 2018
#freebsd #sockstat

The sockstat command list open Internet or UNIX domain sockets.

List open tcp in ipv4:

# sockstat -4l

Show connectd sockets in port 443 and tcp:

# sockstat -P tcp -p 443 -c

List unix sockets:

# sockstat -u
