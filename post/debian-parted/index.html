Sep 26, 2016

Resize AWS Debian Jessie

Probably instance will have 8GB, to extend the disk, for example to 40GB do this:

Install parted:

$ apt-get install parted

Run:

$ /sbin/parted ---pretend-input-tty /dev/xvda resizepart 1 yes 100%

Resize2fs:

$ resize2fs /dev/xvda1

Run lsblk:

root@ip-10-20-1-17:~# lsblk
NAME    MAJ:MIN RM SIZE RO TYPE MOUNTPOINT
xvda    202:0    0  40G  0 disk
`-xvda1 202:1    0  40G  0 part /
