virtualbox-el-capitan

Jul 12, 2016

When trying to install Virtualbox in OS X El capitan you may notice it hangs at Verifying “VirtualBox.pkg”.

You can by pass the warning trying to install via command line with:

$ sudo installer -package /Volumes/VirtualBox/VirtualBox.pkg -target /
