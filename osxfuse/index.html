osxfuse

November 26, 2017
#fuse #sshfs

To install:

brew cask install osxfuse

To install sshfuse:

brew install sshfs

Mount the root volume of remote host:

sshfs bastion:/ /tmp/host -o reconnect,follow_symlinks,volname="LocalRemote"

To unmount:

sudo diskutil umount force /tmp/bastion
