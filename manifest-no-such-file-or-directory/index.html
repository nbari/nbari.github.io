MANIFEST No Such File or Directory

June 12, 2018
#freebsd #MANIFEST #bsdinstall #hetzner

If when trying to install FreeBSD using bsdinstall you get:

"Error while fetching file:///usr/freebsd-dist/MANIFEST" - no such file or directory

Try to:

root@rescue:~ # mkdir -p /usr/freebsd-dist/
root@rescue:~ # touch /usr/freebsd-dist/MANIFEST

And then start again
