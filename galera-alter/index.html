galera alter

January 29, 2019
#galera #mysql #RSU #TOI

To prevent bloking the cluster while doing ALTERS (SCHEMA UPGRADES), do this per node:

SET GLOBAL wsrep_OSU_method='RSU';

Then Run the ALTER statement once done reset the Schema Upgrade method back to Total Order Isolation.

SET GLOBAL wsrep_OSU_method='TOI';
