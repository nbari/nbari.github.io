bash git prompt

March 20, 2018
#linux #bash #prompt

To display the current branch in the terminal prompt using bash, add the following lines to ~/.bash_profile:

parse_git_branch() {
    git branch 2> /dev/null | sed -e '/^[^*]/d' -e 's/* \(.*\)/ (\1)/'
}

export PS1="\u@\h \[\033[32m\]\w\[\033[33m\]\$(parse_git_branch)\[\033[00m\] $ "
